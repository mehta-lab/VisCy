# viscy-utils

:material-tools:{ .lg .middle } Shared ML infrastructure for virtual staining
microscopy.

=== ":material-language-python: pip"

    ```sh
    pip install viscy-utils
    ```

=== ":material-package-variant-closed: uv"

    ```sh
    uv add viscy-utils
    ```

!!! note "Depends on `viscy-data`"

    `viscy-utils` is the one package that builds on another (`viscy-data`); the
    other three are leaf dependencies. See the [dependency layering](index.md#dependency-layering).
