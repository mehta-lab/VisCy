# VisCy

Virtual staining for microscopy.

## Installation

```bash
pip install viscy
```

## Packages

This monorepo contains:

- **viscy-transforms** - Image transforms for virtual staining microscopy

## Development

```bash
# Clone and install
git clone https://github.com/mehta-lab/VisCy.git
cd VisCy
uv sync

# Run tests
uv run pytest

# Run linting
uvx prek run --all-files
```
